## 1. Public Dataset

**Online Retail Dataset**

- Source: UCI Machine Learning Repository
- URL: https://archive.ics.uci.edu/ml/datasets/online+retail
- Description: Transaction-level retail data used as a base to simulate CRM lead events.

This dataset is publicly available and free to use for research and educational purposes.

---

## 2. Synthetic CRM Events

To model real-world CRM behavior, additional lead events are synthetically generated
using custom Python scripts included in this repository.

These events simulate:
- Lead status changes
- Lead re-assignments
- Late arriving data
- Duplicate events

No proprietary or company data is used in this project.
